写在前面：
     类名、Id是使用驼峰法则来写的。

首页
  • 首页主要分这样几个部分：顶部，头图，专栏介绍区，登录后显示区，登录模块和底部。
  • 后端渲染可能涉及的部分为：
    • 登录后用户显示：
      • id personalHead 用户头像
      • id myColHref 我的专栏跳转链接
      • id myColUpload 专栏发布页面链接
    • 专栏介绍区：
      • id recLeft 推荐左专栏
      • id recRight 推荐右专栏
      • class colImg 专栏封面图
      • class owner 用户姓名
      • class colLocation 用户学校
      • class colTitle  专栏名称
      • class colUrl 专栏详情跳转页面链接
      • class colArtical 专栏内容
      • class colDate 专栏创建时间
    • 登录模块
      • id inputPhone 用户手机号
      • id inputPassword 用户密码

专栏
  • 专栏主要分这样几个部分：顶部，头图，专栏介绍区，登录后显示区，登录模块和底部，类名与首页相同。

专栏详情
  • 专栏主要分这样几个部分：顶部，头图，专栏介绍区，登录后显示区，登录模块和底部。
  • 特有字段为：
    • id detailsTitle 专栏名称
    • id detailsOwner 专栏作者
    • id school 用户学校
    • id relationship 查看者与用户关系
    • id detailsDate 专栏创建日期
    • id keys 专栏标签
    • id article 专栏内容
我的专栏
  • 我的专栏主要分上部和专栏展示区。
    • 上部
      • id colHead 用户头像
      • id owner 用户名称
    • 专栏展示区
      • class mycolumnImg 封面图
      • class  mycolumnTitle 专栏标题
      • class mycolumnSum 专栏摘要
      • class mycolDate 专栏日期
上传专栏
  • 上传专栏主要分为图片上传部分和富文本编辑部分两块
    • 图片上传
      • id coverImg 封面图片
      • id imgList 用户已上传图片列表
      • id uploadImg 用户新上传图片
    • 富文本编辑区
      • id uploadTitle 专栏名称
      • id uploadKeys 专栏标签
      • id sum 专栏摘要
      • id myEditor 文章正文
   
页面实现功能
  • 首页 专栏页 
    • ajax实现页面加载专栏
    • ajax提交用户登录信息
  • 我的专栏
    • 删除用户专栏
    • ajax加载专栏(或模版引擎直接渲染)
  • 上传专栏
    • 富文本编辑
    • 图片上传
